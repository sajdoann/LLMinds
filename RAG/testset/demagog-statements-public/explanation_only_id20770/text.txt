<p>V&nbsp;souvislosti s&nbsp;formulováním odpovědi na předběžný audit Evropské komise, který oznamoval existenci střetu zájmů premiéra Babiše, i&nbsp;na jeho konečnou verzi z&nbsp;prosince <a href="https://www.irozhlas.cz/zpravy-domov/evropska-komise-prvni-audit-stret-zajmu-andrej-babis_2104232054_tzr">2019</a>, hovořila ministryně <a href="https://www.novinky.cz/domaci/clanek/ministerstvo-poslalo-do-bruselu-odpoved-na-audit-k-babisove-stretu-zajmu-40325921">Dostálová</a> vždy o&nbsp;tom, že dokument zpracovávají úředníci. Konkrétně uvádí úředníky Národního orgánu pro koordinaci (NOK), jehož <a href="https://kormoran.vlada.cz/usneseni/usneseni_webtest.nsf/web_file/uv130612.0448.doc/$File/uv130612.0448.doc">funkci</a> (.doc, str.&nbsp;1) <a href="https://www.dotaceeu.cz/cs/fondy-eu/narodni-organ-pro-koordinaci">plní</a> Ministerstvo pro místní rozvoj, a&nbsp;jejich „<i>kolegy z dotčených resortů“.</i></p><p>V&nbsp;tomto ohledu zmiňuje server <a href="https://www.irozhlas.cz/zpravy-domov/andrej-babis-stret-zajmu-praha-audit-evropska-komise-agrofert-dotace_1912051210_kno">iRozhlas</a> součást předběžné auditní zprávy z&nbsp;června 2019, kdy uvádí, že se jedná o&nbsp;„<i>čerpání dotací ze strukturálních fondů a&nbsp;fondu soudržnosti. Jde tedy o&nbsp;peníze, které přerozděluje především ministerstvo průmyslu a&nbsp;obchodu a&nbsp;ministerstvo práce a&nbsp;sociálních věcí.“</i></p><p>V&nbsp;závěrečné verzi auditní zprávy, kterou Evropská komise <a href="https://ec.europa.eu/regional_policy/sources/docoffic/official/reports/cz_functioning_report/cz_functioning_report_en.pdf">zveřejnila</a> (.pdf, str.&nbsp;1–2) letos v&nbsp;dubnu, si však můžeme přečíst i&nbsp;seznam institucí „<i>zapojených do procesu řízení a&nbsp;rozdělování dotací“</i>, které byly předmětem auditu, a&nbsp;které jsou tedy výše jmenovanými „<i>dotčenými resorty“</i>. Právě tyto instituce tedy komunikovaly s&nbsp;Evropskou komisí.</p><p>Kromě nadresortní <a href="https://www.dotacni.info/tag/spolecny-strategicky-ramec/">Rady</a> pro fondy Společného strategického rámce, <a href="https://www.dotaceeu.cz/cs/evropske-fondy-v-cr/narodni-organ-pro-koordinaci/pracovni-skupiny/rada-pro-fondy-ssr">Rady</a> pro ESI fondy a&nbsp;Řídícího a&nbsp;koordinačního <a href="https://www.dotaceeu.cz/cs/evropske-fondy-v-cr/narodni-organ-pro-koordinaci/koordinace/ridici-a-koordinacni-vybor">výboru</a> se jedná o&nbsp;již zmíněný Národní orgán pro koordinaci. Jsou zde také uvedena tři ministerstva: Ministerstvo práce a&nbsp;sociálních věcí, Ministerstvo průmyslu a&nbsp;obchodu, Ministerstvo životního prostředí. Mimo ně zpráva uvádí také Agenturu pro podnikání a&nbsp;inovace, Státní fond životního prostředí České republiky, Agenturu pro podporu podnikání a&nbsp;investic CzechInvest a&nbsp;hlavní město Prahu.&nbsp;</p><p>Celkem auditní zpráva <a href="https://ec.europa.eu/regional_policy/sources/docoffic/official/reports/cz_functioning_report/cz_functioning_report_en.pdf">vyjmenovává</a> (.pdf, str.&nbsp;1–2) 7 „<i>řídících a&nbsp;zprostředkujících institucí“</i>, mezi něž spadají právě uvedená ministerstva a&nbsp;agentury, a&nbsp;4 „<i>horizontální instituce“</i>, kam patří Národní orgán pro koordinaci a&nbsp;další nadresortní instituce.</p><p>Ministryně Dostálová má pravdu také v&nbsp;tom, že se nejedná pouze o&nbsp;ministerstva pod vedením ministrů za hnutí ANO. Ministerstvo práce a&nbsp;sociálních věcí vede <a href="https://www.mpsv.cz/ministryne">Jana Maláčová</a> z&nbsp;ČSSD a&nbsp;primátorem Prahy je <a href="https://www.praha.eu/jnp/cz/o_meste/primator_a_volene_organy/Primator/vizitka_primatora_hmp/index.html">Zdeněk Hřib</a> z Pirátské strany.</p><p><br>&nbsp;</p>
