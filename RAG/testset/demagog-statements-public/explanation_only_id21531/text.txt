<p>Bývalá vláda <a href="https://www.vlada.cz/cz/media-centrum/aktualne/vlada-schvalila-podpurne-programy-pro-podnikatele-postizene-protiepidemickymi-opatrenimi-192462/">představila</a> během pandemie covidu-19 několik programů na pomoc zaměstnancům, firmám a&nbsp;osobám samostatně výdělečně činným (OSVČ).</p><p>Co se týče nynějších <a href="https://covid.gov.cz/situace/kompenzace">kompenzačních</a> programů, pod Ministerstvo práce a&nbsp;sociálních věcí <a href="https://www.mpo.cz/cz/rozcestnik/informace-o-koronavirus/covid-dotacni-rozcestnik--259263/">spadají</a> například programy <a href="https://covid.gov.cz/situace/kompenzace/antivirus-nucene-omezeni-provozu-karanteny">Antivirus&nbsp;A</a> a&nbsp;<a href="https://covid.gov.cz/situace/kompenzace/antivirus-b">Antivirus&nbsp;B</a>. Účelem prvního z&nbsp;nich jsou <a href="https://www.vlada.cz/cz/epidemie-koronaviru/dulezite-informace/podpora-a-ulevy-pro-podnikatele-a-zamestnance-180601/">kompenzace</a> nákladů zaměstnavatelům, jejichž zaměstnanci měli nařízenou karanténu nebo izolaci. U&nbsp;Antiviru&nbsp;B se pak jednalo o&nbsp;<a href="https://www.vlada.cz/cz/epidemie-koronaviru/dulezite-informace/podpora-a-ulevy-pro-podnikatele-a-zamestnance-180601/">kompenzace</a> firmám v&nbsp;případě, že musely omezit provoz kvůli významnému počtu zaměstnanců v&nbsp;karanténě, nebo například je-li z&nbsp;důvodu pandemie omezená poptávka po jejich službách či produktech.</p><p>Pro poškozené OSVČ, ale také pro společníky malých firem a&nbsp;osoby pracující na DPČ a&nbsp;DPP je zde program <a href="https://www.mfcr.cz/cs/aktualne/tiskove-zpravy/2021/od-22-listopadu-bude-pokracovat-kompenza-43629">Kompenzační bonus</a>, který <a href="https://www.mpo.cz/cz/rozcestnik/informace-o-koronavirus/covid-dotacni-rozcestnik--259263/">spadá</a> pod Ministerstvo financí.</p><p>V&nbsp;<a href="https://www.mpo.cz/covid-2021">gesci</a> Ministerstva průmyslu a&nbsp;obchodu měly být například dotační program <a href="https://covid.gov.cz/situace/kompenzace/covid-2021">COVID 2021</a> a&nbsp;program <a href="https://covid.gov.cz/situace/kompenzace/covid-nepokryte-naklady">COVID Nepokryté náklady</a>, <a href="https://www.mpo.cz/cz/rozcestnik/pro-media/tiskove-zpravy/nova-podoba-kompenzaci-k-programu-na-nepokryte-naklady-vlada-schvalila-take-program-covid---2021-s-podporou-500-kc-za-den-a-zamestnance---259983/">určené</a> pro firmy s&nbsp;významným poklesem tržeb. Přestože byly již připravené, vláda nakonec rozhodla, že je opětovně <a href="https://www.podnikatel.cz/aktuality/firmy-maji-smulu-novy-ministr-odmitl-programy-covid-2021-a-nepokryte-naklady/">nespustí</a>.</p><p>Doplňme, že v&nbsp;současné době také fungují některé programy, jejichž účelem je primárně poskytnout podnikatelům <a href="https://covid.gov.cz/situace/kompenzace">záruky</a> za úvěry, nejedná se nicméně o&nbsp;klasické kompenzace. Zmínit můžeme např.&nbsp;záruční program <a href="https://covid.gov.cz/situace/kompenzace/zaruka-covid-sport">COVID Sport</a> spadající pod Ministerstvo průmyslu a&nbsp;obchodu. V&nbsp;jeho gesci byl také nedávno ukončený záruční program <a href="https://covid.gov.cz/situace/kompenzace/covid-zaruka-ck-prijem-zadosti-ukoncen">COVID Záruka CK</a> pro cestovní kanceláře, na který navázal program <a href="https://www.nrb.cz/zaruky-cestovnim-kancelarim-bude-nrb-poskytovat-i-v-roce-2022/">Záruka CK</a> Ministerstva pro místní rozvoj.</p><p>Z&nbsp;výše uvedených informací vyplývá, že kompenzační programy se skutečně dělí především mezi tři zmíněná ministerstva. Výrok Mariana Jurečky proto hodnotíme jako pravdivý.</p>
